- Styled Components

## Як користуватися цим репозиторієм

1. Клонуйте репозиторій: `git clone https://github.com/IG00RA/portfolio.git`
2. Встановіть залежності: `npm install`
3. Запустіть локальний сервер: `npm run dev`
